Color Contrast

Serious AA Perceivable

The visual presentation of text and images of text has a contrast ratio of at least 4.5:1 for normal text and 3:1 for large text. This ensures that people with moderately low vision can read the text without assistive technology.

2 element(s) affected Page: Accessibility Demonstration Page
üìã Detailed Information ‚ñº

User Impact

Users with low vision, color blindness, or using devices in bright environments may not be able to read the content

WCAG Reference

1.4.3 Contrast (Minimum)

Common Mistakes

  • Using only color to convey information
  • Insufficient contrast on interactive elements
  • Not testing with different color blindness types
  • Overlooking contrast in hover/focus states
üîß How to Fix ‚ñº
  1. Check the contrast ratio between text and background colors
  2. Use a contrast ratio of at least 4.5:1 for normal text
  3. Use a contrast ratio of at least 3:1 for large text (18pt or 14pt bold)
  4. Test with color contrast analyzers
  5. Consider users with color vision deficiencies

Images must have alternative text

Critical A Perceivable

Images that convey information must have alternative text that serves the equivalent purpose. Decorative images should have empty alt attributes (alt="") to be ignored by screen readers.

1 element(s) affected Page: Accessibility Demonstration Page
üìã Detailed Information ‚ñº

User Impact

Screen reader users cannot understand image content without proper alternative text

WCAG Reference

1.1.1 Non-text Content

Common Mistakes

  • Missing alt attributes entirely
  • Using filename as alt text
  • Alt text that doesn't match image content
  • Not marking decorative images properly
üîß How to Fix ‚ñº
  1. Add descriptive alt text to informative images
  2. Use alt="" for decorative images
  3. Include alt text that conveys the same information as the image
  4. Keep alt text concise but descriptive
  5. Avoid phrases like "image of" or "picture of"
